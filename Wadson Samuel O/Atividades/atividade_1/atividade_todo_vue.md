# 📝 Atividade: Refatoração To-Do com Vue 3

## 🎯 Objetivo
Refatorar o aplicativo de lista de tarefas desenvolvido em JavaScript puro para utilizar **Vue 3**, com foco no uso da diretiva **v-for**.

---

## 📅 Entrega
**Data:** 23/04  
**Forma de entrega:** GitHub  

### 📁 Estrutura obrigatória
Criar dentro do repositório a pasta:

```
atividades/atividade_todo_vue
```

---

## 📌 Requisitos

Recriar o To-Do mantendo:

- Adicionar tarefa  
- Listar tarefas  
- Marcar como concluída  
- Remover tarefa  

---

## ⚠️ Regras

**Não usar:**
- `document.getElementById`
- `createElement`
- `addEventListener`

**Usar:**
- Vue 3 (CDN)  
- `v-model`  
- `@click`  
- **v-for (obrigatório)**  

---

## 📚 Documentação

Estudar:

https://vuejs.org/guide/essentials/list.html

---

## Foco da atividade

- Uso correto do `v-for`  
- Manipulação de listas com Vue  
- Atualização automática da interface via dados  

---

##  Critérios de avaliação

- Uso correto do `v-for`  
- Funcionalidade completa  
- Sem manipulação manual do DOM  
- Organização básica do código
